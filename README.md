# Ride Sharing Analysis Project

## Overview
This project simulates a **ride‑sharing platform database** and demonstrates advanced SQL analytics.  
It covers **database design, data insertion, and business insights** using queries with joins, aggregations, and window functions.  
The goal is to showcase **data analytics and SQL mastery** for portfolio and interview preparation.

---

##  Database Schema
The project consists of **7 tables**:

- **Drivers** → Driver details, ratings, and completed rides  
- **Customers** → Customer demographics and loyalty status  
- **Vehicles** → Vehicle information linked to drivers  
- **Payments** → Payment methods, amounts, and statuses  
- **Locations** → Pickup and drop points across cities  
- **Rides** → Ride transactions with fares, distances, and ratings  
- **RideFeedback** → Customer feedback and ratings per ride  

---

##  Sample Data
- **Drivers**: 13 drivers with varied ratings and ride counts  
- **Customers**: 11 customers with loyalty tiers (Gold, Silver, Bronze, Platinum)  
- **Vehicles**: Cars and bikes assigned to drivers  
- **Payments**: Card, Cash, Wallet, UPI with success/failure cases  
- **Locations**: Popular pickup/drop points across Mumbai, Delhi, Bangalore, Kolkata  
- **Rides**: 50+ rides with realistic fares, distances, and timestamps  
- **Feedback**: Customer comments and ratings  

---

##  Key SQL Queries & Insights
- **Top Drivers by Earnings per City** → RANK() function  
- **High‑Spending Customers** → Subquery vs average spending  
- **Driver with Maximum Rides per City** → ROW_NUMBER() function  
- **Driver Performance Classification** → IF conditions (Excellent, Good, Needs Improvement)  
- **Top 5 Spending Customers** → Loyalty + spending analysis  
- **Driver Performance Analysis** → Avg fare per ride + total rides  
- **Popular Pickup–Drop Pairs** → Route frequency and revenue  
- **Customers Without Feedback** → LEFT JOIN analysis  
- **Top Paying Customers per City** → RANK() partitioned by city  
- **Ride Volume Trends** → Monthly ride counts vs average  

---

##  Business Applications
- **Driver Leaderboards** → Incentives and performance tracking
